About the project

This is a personal project by @GreekFellows to write a beginner's book for Japanese people interested in learning Old Norse, an ancient Scandinavian language.

The only three reasons I am writing this book is because

  • Old Norse is amazingly fun;
  • there are not many learning resources written in Japanese; and
  • I can revisit concepts as I write the book.

How to get the book

If you have git, simply clone this repository. If you have not, you can download the repository as a ZIP from GitHub by clicking on this link.

The book is written in LaTeX. The repository includes both the source file .tex and a compiled PDF .pdf.
