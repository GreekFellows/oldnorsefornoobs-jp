これはなに？

12 世紀頃にスカンディナヴィア地方で話されていた、古ノルド語という言語を勉強したい日本人の方のために @GreekFellows が書いている入門書です。

現在、本の構成を考えています。

何故入門書を書くの？

  • 古ノルド語が面白いので、色んな人に知って欲しいから。
  • 日本語で書かれた古ノルド語の教材があまりにもないから。
  • 教材を書くうちに、自分も知識を整理することが出来るから。

本を読むには？

git があれば、このレポジトリを clone してください。git がないのなら、このレポジトリを ZIP ファイルとしてこちらのリンクからダウンロード出来ます。

この本は LaTeX で書かれています。レポジトリには、ソースである .tex ファイルとコンパイル済みの .pdf が両方用意されています。

About the project

This is a personal project by @GreekFellows to write a beginner's book for Japanese people interested in learning Old Norse, an ancient Scandinavian language.

The book is still in her planning process.

Why write a book?

  • Old Norse is amazingly fun;
  • there are not many learning resources written in Japanese; and
  • I can revisit concepts as I write the book.

How to get the book

If you have git, simply clone this repository. If you have not, you can download the repository as a ZIP from GitHub by clicking on this link.

The book is written in LaTeX. The repository includes both the source file .tex and a compiled .pdf.
